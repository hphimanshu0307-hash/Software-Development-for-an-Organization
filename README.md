# ERP System Development

## Overview
Built and deployed a full ERP system for ICS India Private Limited, which is a 23 year old organization. Currently used by 300 active users across departments.

## Problem
- Manual processes
- Data silos
- Poor visibility for decision-making
- Scaling issues

## Solution
- A Centralized system
- Modular system: HR, Marketing, Finance, Operations
- Role-based access and workflows

## AI Enhancement

- Reduced 13 inefficiencies

## Impact
